# Board Template #2 - Ornate Frame Chalkboard

## Description
Elegant ornate frame design with decorative border image. Features classic chalkboard aesthetic with beautiful typography.

## Features
- Ornate decorative border using border-image
- 100px border width (responsive)
- Black chalkboard background
- 2-column layout
- Olive/lime green headings (#999D15)
- Custom fonts (Brilon, Poppins)
- Letter-spaced typography

## Usage
```
[chalkboard_menu_pro template="2"]
```

## Files
- `style.css` - Template-specific stylesheet
- `menu-border-small.png` - Ornate frame border image
- `back-big-blackboard-BW.jpg` - Chalkboard background texture
- `mainbg.png` - Main background image
- `footerbg.png` - Footer background image

## Border Image Settings
- Border width: 100px (desktop), 75px (tablet), 50px (mobile)
- Border image slice: 116 110 117 111
- Border image repeat: repeat

## Customization
Edit `style.css` to customize:
- Border width and image slicing
- Heading color and size
- Font families
- Letter spacing
- Column layout
- Background colors
- Responsive breakpoints

## Typography
- Headings: 30px, 5px letter-spacing
- Items: 18px, 2px letter-spacing
- Font: Brilon, Poppins (with sans-serif fallback)

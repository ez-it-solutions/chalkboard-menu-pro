# Board Template #1 - Simple Chalkboard

## Description
Clean, modern chalkboard design with rounded corners and simple border styling. Perfect for minimalist displays.

## Features
- Rounded corners (12px border-radius)
- Simple shadow border effect
- 3-column grid layout
- Radial gradient background
- Olive/lime green headings (#c4cf4d)
- Serif font (Georgia, Times New Roman)

## Usage
```
[chalkboard_menu_pro template="1"]
```

## Files
- `style.css` - Template-specific stylesheet
- `preview.png` - Template preview image

## Customization
Edit `style.css` to customize:
- Colors (headings, text, background)
- Fonts and typography
- Border styling
- Grid layout (columns, gaps)
- Responsive breakpoints
